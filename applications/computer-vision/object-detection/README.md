# Методы обнаружения объектов (Object Detection)

## Введение

Обнаружение объектов - это одна из ключевых задач компьютерного зрения, которая заключается в определении местоположения и классификации объектов на изображениях или видео. В отличие от классификации изображений, где мы определяем, что изображено на картинке целиком, в задаче обнаружения объектов мы должны найти все объекты интереса и определить их координаты (обычно в виде ограничивающих прямоугольников).

## Основные подходы к обнаружению объектов

### 1. Традиционные методы
- Каскадные классификаторы Хаара
- HOG (Histogram of Oriented Gradients) + SVM
- SIFT/SURF дескрипторы

### 2. Нейросетевые методы
- R-CNN, Fast R-CNN, Faster R-CNN
- YOLO (You Only Look Once)
- SSD (Single Shot Detector)
- RetinaNet

## Применение

Обнаружение объектов используется в различных приложениях:
- Системы видеонаблюдения
- Автономные автомобили
- Распознавание лиц
- Медицинская диагностика
- Робототехника
- Дополненная реальность

## Сравнение подходов

| Метод | Преимущества | Недостатки | Скорость | Точность |
|-------|--------------|------------|----------|----------|
| Каскадный классификатор Хаара | Простота, высокая скорость | Низкая точность, чувствителен к освещению | Высокая | Низкая |
| YOLO | Высокая точность, реальное время | Сложность реализации | Средняя | Высокая |
| R-CNN | Высокая точность | Медленная работа | Низкая | Высокая |

В этом разделе мы рассмотрим различные методы обнаружения объектов, начиная с классических подходов и заканчивая современными нейросетевыми решениями.
