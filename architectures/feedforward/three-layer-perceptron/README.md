# Архитектура трехслойного персептрона (3-layer Perceptron)

> **Примечание**: В этом каталоге также содержится [реализация на Python](mlp.py) трехслойного персептрона с подробными комментариями и примерами использования.

## Введение

Трехслойный персептрон, также известный как многослойный персептрон (Multilayer Perceptron, MLP) с одним скрытым слоем, представляет собой развитие оригинальной модели персептрона Розенблатта. Эта архитектура преодолевает ключевое ограничение оригинального персептрона - невозможность решать задачи, не являющиеся линейно разделимыми, такие как проблема XOR.

> **См. также**: [Персептрон Розенблатта](../../foundations/neuron-models/rosenblatt/) для понимания предшествующей модели. См. также [Сравнение моделей](../../foundations/neuron-models/comparison.md) для анализа различий между подходами.

## Почему это важно?

Прежде чем углубляться в технические детали, давайте поймем, почему трехслойный персептрон был таким важным. Представьте, что вы хотите создать программу, которая может распознавать рукописные цифры. Оригинальный персептрон не мог бы справиться с такой задачей, потому что форма цифр не является линейно разделимой. Трехслойный персептрон же может "учиться" представлять сложные нелинейные зависимости, автоматически подстраивая свои параметры - это как учить ребенка различать сложные формы, показывая ему много примеров.

## Исторический контекст

Трехслойный персептрон стал развитием оригинальной модели персептрона после того, как Marvin Minsky и Seymour Papert в своей книге "Perceptrons" (1969) показали фундаментальные ограничения однослойных персептронов. Это привело к "зиме искусственного интеллекта" в 1970-х годах, пока в 1980-х не был разработан алгоритм обратного распространения ошибки, который позволил эффективно обучать многослойные сети.

## Основные принципы архитектуры

### Структура трехслойного персептрона

1. **Входной слой**: Получает входные сигналы
2. **Скрытый слой**: Промежуточные элементы с нелинейными активациями
3. **Выходной слой**: Производит окончательный результат

### Математическое описание

Функционирование трехслойного персептрона можно описать следующим образом:

```
Скрытый слой: h = σ(W₁x + b₁)
Выходной слой: y = σ(W₂h + b₂)
```

Где:
- x - входной вектор
- W₁, W₂ - матрицы весов
- b₁, b₂ - векторы смещений
- σ - нелинейная функция активации (например, сигмоида)
- h - активации скрытого слоя
- y - выходной вектор

### Функции активации

В отличие от оригинального персептрона Розенблатта, который использует пороговую функцию активации, трехслойный персептрон использует нелинейные функции активации:

1. **Сигмоида**: σ(x) = 1 / (1 + e^(-x))
   - Преимущества: Гладкая, дифференцируемая функция
   - Недостатки: **Проблема затухающих градиентов (vanishing gradient problem)** - производная сигмоиды максимальна (0.25) при x=0 и быстро стремится к нулю при удалении от нуля. При обратном распространении ошибки через много слоев произведение таких маленьких производных становится экспоненциально малым, практически останавливая обучение ранних слоев

2. **ReLU (Rectified Linear Unit)**: f(x) = max(0, x)
   - Преимущества: Вычислительно эффективна, помогает решить проблему затухающих градиентов
   - Недостатки: "Мертвые" нейроны

3. **Гиперболический тангенс**: tanh(x) = (e^x - e^(-x)) / (e^x + e^(-x))
   - Преимущества: Значения центрированы вокруг нуля (диапазон [-1, 1]), что может ускорить обучение
   - Недостатки: Также страдает от проблемы затухающих градиентов, хотя в меньшей степени, чем сигмоида (максимальная производная 1.0 вместо 0.25)

### Алгоритм обратного распространения ошибки

Алгоритм обратного распространения ошибки (backpropagation) состоит из двух фаз:

1. **Прямое распространение**:
   - Входные данные проходят через сеть слой за слоем
   - На каждом нейроне вычисляется активация
   - Получается предсказание сети

2. **Обратное распространение**:
   - Вычисляется ошибка между предсказанием и целевым значением
   - Ошибка распространяется назад через сеть
   - Обновляются веса с использованием градиентного спуска

Математически:
```
∂E/∂w_ij = ∂E/∂out_j * ∂out_j/∂net_j * ∂net_j/∂w_ij
```

Где:
- E - функция ошибки
- out_j - выход j-го нейрона
- net_j - взвешенная сумма входов j-го нейрона
- w_ij - вес связи между i-м и j-м нейронами

### Визуальное представление

```
Входной слой          Скрытый слой          Выходной слой
┌─────────────┐       ┌─────────────┐       ┌─────────────┐
│    x₁       │──────▶│    h₁       │──────▶│    y₁       │
├─────────────┤  ┌───▶├─────────────┤  ┌───▶├─────────────┤
│    x₂       │──┤    │    h₂       │──┤    │    y₂       │
├─────────────┤  ├───▶├─────────────┤  ├───▶├─────────────┤
│    x₃       │──┘    │    h₃       │──┘    │    y₃       │
├─────────────┤       ├─────────────┤       └─────────────┘
│    ...      │       │    ...      │
├─────────────┤       ├─────────────┤
│    xₙ       │       │    hₘ       │
└─────────────┘       └─────────────┘
```

### Конкретный пример

Рассмотрим трехслойный персептрон для решения задачи XOR:

```
Входной слой: 2 нейрона (для двух входов)
Скрытый слой: 2 нейрона с сигмоидальной активацией
Выходной слой: 1 нейрон с сигмоидальной активацией

Веса между входом и скрытым слоем:
W₁ = [[10, -10], 
      [10, -10]]

Веса между скрытым и выходным слоем:
W₂ = [[10], 
      [10]]

Смещения:
b₁ = [-5, 5]
b₂ = [-5]

Вычисления для входа [0, 0]:
Скрытый слой: 
h₁ = σ(0*10 + 0*(-10) - 5) = σ(-5) ≈ 0.007
h₂ = σ(0*10 + 0*(-10) + 5) = σ(5) ≈ 0.993

Выходной слой:
y = σ(0.007*10 + 0.993*10 - 5) = σ(5) ≈ 0.993 ≈ 1 (неправильно!)

После обучения веса будут скорректированы для правильной работы.
```

### Реализация на Python

В файле [mlp.py](mlp.py) содержится полная реализация трехслойного персептрона на языке Python с использованием только NumPy. Реализация включает:

1. Класс `ThreeLayerPerceptron` с методами прямого и обратного распространения
2. Поддержку различных функций активации (сигмоида)
3. Методы обучения и предсказания
4. Примеры использования для задачи XOR, аппроксимации функции и классификации точек внутри/вне круга

Для запуска примеров выполните:

```bash
python mlp.py
```

Реализация демонстрирует ключевые концепции обучения многослойных сетей, включая:
- Инициализацию весов методом Xavier/Glorot
- Прямое распространение сигнала
- Обратное распространение ошибки
- Обновление весов градиентным спуском

Кроме того, реализация включает визуализацию:
- Границ принятия решений для классификационных задач
- Процесса обучения (графики ошибок)
- Аппроксимации функций

### Особенности архитектуры

1. **Многослойность**: Наличие скрытого слоя позволяет решать нелинейные задачи
2. **Нелинейные активации**: Использование нелинейных функций активации (сигмоида, ReLU и др.)
3. **Полносвязность**: Каждый нейрон в одном слое соединен со всеми нейронами в следующем слое
4. **Обратное распространение**: Алгоритм обучения с использованием градиентного спуска
5. **Универсальная аппроксимация**: Может аппроксимировать любую непрерывную функцию при достаточном количестве нейронов

### Преимущества и недостатки

**Преимущества**:
- Способность решать нелинейно разделимые задачи
- Относительная простота реализации
- Хорошо изучены теоретически
- Эффективны для широкого круга задач

**Недостатки**:
- **Проблема затухающих градиентов** - при обратном распространении ошибки через много слоев градиенты становятся экспоненциально маленькими в ранних слоях из-за умножения маленьких производных активационных функций, что практически останавливает обучение ранних слоев. Это особенно критично при использовании сигмоиды или tanh активаций и при обучении глубоких сетей (более 3-4 слоев)
- Риск переобучения, особенно при большом количестве параметров относительно размера обучающей выборки
- Чувствительность к начальным весам - неправильная инициализация может привести к проблемам с обучением или медленной сходимости
- Требуют большого количества данных для обучения глубоких сетей
- Медленная сходимость в некоторых случаях, особенно при использовании сигмоидальных активаций

## Гиперпараметры и настройка обучения

При обучении трехслойного персептрона необходимо правильно выбрать гиперпараметры:

1. **Скорость обучения (learning rate)**:
   - Слишком высокая: нестабильное обучение, возможна расходимость
   - Слишком низкая: медленная сходимость
   - Рекомендуемые значения: 0.001 - 0.1

2. **Количество нейронов в скрытом слое**:
   - Слишком мало: недообучение
   - Слишком много: переобучение, увеличение времени обучения
   - Эмпирическое правило: от числа входов до числа выходов

3. **Количество эпох**:
   - Должно быть достаточным для сходимости
   - Слишком много: риск переобучения
   - Используйте раннюю остановку (early stopping)

4. **Размер батча**:
   - Полный батч: стабильное, но медленное обучение
   - Мини-батчи: компромисс между стабильностью и скоростью
   - Стохастический градиентный спуск: быстрое, но шумное обучение

## Регуляризация

Для предотвращения переобучения применяются методы регуляризации:

1. **L1/L2 регуляризация**: добавление штрафа за большие веса в функцию потерь
2. **Dropout**: случайное отключение части нейронов во время обучения
3. **Ранняя остановка**: прекращение обучения при ухудшении качества на валидационной выборке
